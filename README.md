## Local Development

This is a static site generated from a Nuxt 3 application. To make changes:

1. Clone the source repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Generate static files: `npm run generate`
5. Deploy the `.output/public` folder to this repository
